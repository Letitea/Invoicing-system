## 系統說明

專案運行方式  
step 1. clone code  
step 2. npm i  
step 3. npm run start  

Node.js 版本 20.9.0  
Angular Cli 版本 17.0.3

## 資料夾說明
src > app > components 放置系統畫面所需之component  
src > app > models 放置定義的model  
src > app > services 放置抓取JSON之程式  
src > assets 放置文字檔、圖檔、JSON檔
